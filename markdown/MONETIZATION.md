## üí° Conversion Optimization

### Free ‚Üí Premium Triggers

1. **Hit Monthly Limit**
   - Show usage banner when approaching limit
   - Block upload when limit reached
   - Immediate upgrade prompt

2. **File Too Large**
   - "Your file is 40MB. Upgrade for 100MB limit!"
   - Show size comparison

3. **Need More Ranges**
   - "Unlock unlimited ranges for $2/month"
   - Show feature comparison

4. **ZIP Downloads**
   - Tease feature when they have multiple slices
   - "Download all as ZIP with Premium"

5. **Ad Frustration**
   - "Remove ads for just $2/month"
   - Strategic ad placement

### Upgrade Prompt Timing
- After 2nd PDF upload
- When adding 3rd page range
- After successful slice (success state)
- Monthly reminder email
